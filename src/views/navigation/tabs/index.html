Tabs

Links placed loose inside of a nav element will simply wrap as inline text might: 

<nav>
  <a href="#">Loose link inside of nav element</a>
  <a href="#">Another loose link</a>
  <a href="#">A third example</a>
</nav>

Tabs

Where anchors are inside a paragraph tag, the navigation will form a set of tabs. 

<nav>
  <p>
    <a href="#">Tab navigation</a>
    <a href="#">Another tab level navigation item</a>
    <a href="#">A third example</a>
    <em>Current tab</em>
  </p>
</nav>

Carousel tabs

If a large number of tabs are required, carousel markup can ensure they all fit in the viewport. 

<nav>
  <p role="tablist">
    <a href="#" role="tab">Tab</a>
    <a href="#" role="tab">Tab</a>
    <em role="tab">Current tab</em>
  </p>
</nav>
