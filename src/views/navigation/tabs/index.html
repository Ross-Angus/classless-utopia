Tabs

Links placed loose inside of a nav element will simply wrap as inline text might: 

<nav>
  <a href="#">Loose link inside of nav element</a>
  <a href="#">Another loose link</a>
  <a href="#">A third example</a>
</nav>

Tabs

Where anchors or buttons have the appropriate ARIA attributes, the navigation will form a set of tabs.

The markup pattern below will be updated using JavaScript to add additional ARIA attributes. This markup pattern is taken from the ARIA Authoring Practices Guide.

Markup

Using fragment links

<p role="tablist">
  <a href="#tab1" role="tab">First tab</a>
  <a href="#tab2" role="tab">Second tab</a>
  <a href="#tab3" role="tab">Third tab</a>
</p>

<div role="tabpanel" id="tab1">
  <p>This is tab 1</p>
</div>
<div role="tabpanel" id="tab2">
  <p>This is tab 2</p>
</div>
<div role="tabpanel" id="tab3">
  <p>This is tab 3</p>
</div>

Using buttons

<p role="tablist">
  <button type="button" aria-controls="tabpanel-a" role="tab">
    First tab
  </button>
  <button type="button" aria-controls="tabpanel-b" role="tab">
    Second tab
  </button>
  <button type="button" aria-controls="tabpanel-c" role="tab">
    Third tab
  </button>
</p>
<div role="tabpanel" id="tabpanel-a">
  <p>This is tab 1</p>
</div>
<div role="tabpanel" id="tabpanel-b">
  <p>This is tab 2</p>
</div>
<div role="tabpanel" id="tabpanel-c">
  <p>This is tab 3</p>
</div>
