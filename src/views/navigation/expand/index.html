Expanding navigation

Use of a details tag allows for a collapsing vertical style navigation. This might be useful for in-page navigation, for example a table of contents.

<nav>
  <details open>
    <summary>Jump to:</summary>
    <ul>
      <li><a href="#">Loose link inside of nav element</a>
        <ul>
          <li><a href="#">Aenean sed orci purus</a></li>
        </ul>
      </li>
    </ul>
  </details>
</nav>
