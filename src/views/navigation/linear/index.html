Classless Utopia - Linear navigation

Linear navigation

Links placed loose inside of a nav element will simply wrap as inline text might:

<nav>
  <a href="#">Loose link inside of nav element</a>
  <a href="#">Another loose link</a>
  <a href="#">A third example</a>
</nav>

Example

Loose link inside of nav element Another loose link A third example Aenean sed orci purus Donec et volutpat magna Quisque nulla justo, convallis eget posuere tempor, ultrices nec nisl Sed sollicitudin velit ac metus vestibulum mollis Fusce et tristique nulla

Justified example

A similar pattern, but with the links inside of a paragraph tag gives a more structured layout:

<nav>
  <p>
    <a href="#">Top level navigation</a>
    <a href="#">Another top level navigation item</a>
    <a href="#">A third example</a>
  </p>
</nav>
