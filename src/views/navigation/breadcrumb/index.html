Breadcrumb

Breadcrumb or crumb trails are used to show the hierarchy of the site, reflecting the vertical depth of the navigation tree.

Markup

<nav aria-label="Breadcrumb">
  <ol>
    <li><a href="/">Home</a></li>
    <li><a href="#">Top level section</a></li>
    <li><a href="#">Sub-level section</a></li>
    <li><em title="You are here">The current page</em></li>
  </ol>
</nav>

Accessibility

The functionality and features of the breadcrumb drew inspiration from work by NN/g and ARIA Authoring Practices Guide. However, the NN/g article suggests avoiding duplicating the home link on both the main navigation and the breadcrumb. Later, it suggests hiding long breadcrumbs from mobile. This was not implemented.

The aria-label attribute is used to label the crumb trail. This was not used as a CSS selector.
