Pagination

Simple anchor and button tags are granted the status of a call-to-action button by their proximity to other specific types of inputs and anchors.

The CSS looks for elements which are associated with pagination, specifically:

<a href="/" rel="start">Start</a>
<a href="/" rel="prev">Prev</a>
<button type="button" rel="first">First</button>
<button type="button" rel="previous">Previous</button>

Any following siblings (using either a button or a tag) will be styled to match the call-to-action button, but without the chevron.
