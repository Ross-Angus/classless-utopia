Figures

Figures can be highlighted using the figure and data tags.

Markup

<figure>
  <p><data value="1000">1,000</data></p>
  <figcaption>A big number with no unit</figcaption>
</figure>

Tints

The tints against the background are determined by three values which appear in the value attribute of the data tag: ., %, and £. The presence of each applies a different tint to the background.
