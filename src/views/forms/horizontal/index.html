Horizontal layout

Placing form elements inside a paragraph tag which is an immediate child of a fieldset or form element allows them to take a horizontal layout. For example:

<fieldset>
  <legend>
    <label for="search">Search</label>
  </legend>
  <p>
    <input name="q" id="search" type="search">
    <button>Search</button>
  </p>
</fieldset>

In the above example, the input and button tags are in a horizontal flexbox layout with the input element taking up the remaining space, after the button width has been calculated. This can work in a similar way to media object.
