Dialog

The dialog tag renders a modal popup to the page. This type of control removes all distractions from the page and encourages the user to focus on only one task. It is usually styled with a box in the centre of the screen with the rest of the page behind a semi-transparent veil.

First example dialog

By default, dialog elements are only as wide as their contents.

Second example dialog

Once text inside a dialog element wraps, the dialog element stretches to the full content width of the page.
