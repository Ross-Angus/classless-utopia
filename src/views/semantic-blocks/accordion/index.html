Detail element attributes

Open

The open attribute can be added to a details element in order to make it open by default. It is similar to the disabled attribute on a form element in that it has no value. The presence of the attribute with any value will open the details element when the page is first rendered.

In order to stop this functionality, the attribute should be removed. Changing the value of the attribute to another value such as false will have no effect.

Name

The name attribute can be added to a details element in order to make an explicit connection between it and another attribute on the same page. This works in a similar way to how groups of radio buttons can be linked together so that only one can be selected at any time.

ID

Of course, the details element can also have an ID attribute.
