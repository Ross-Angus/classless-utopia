Carousel

Introduction

Native CSS carousels are only supported in Chrome at the time of writing. The technique used here is heavily based on the CSS Tricks article.

If the technique is not supported, a warning message is displayed and the browser shows a simple scrolling box instead.

Markup

This markup is based on the pattern demonstrated in the ARIA Authoring Practices Guide (APG).

The standard grid markup is used which means that each slide could be any valid containing element such as div, article, aside, etc.

<div aria-roledescription="carousel">
  <section role="group" aria-roledescription="slide">
    Slide content
  </section>
  <section role="group" aria-roledescription="slide">
    Slide content
  </section>
  ...
</div>

The individual slides do not need to be section tags - it's also possible for them to be div tags. Naturally, you can continue to use column layouts inside:
