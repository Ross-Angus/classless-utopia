Classless Utopia - Typography

Typography

The width of the site is determined by the CSS variable --site-width. However, this is not the ideal content width as the line length will be too long for comfortable reading.

Elements such as article, when used as the direct child of main will use the --content-width CSS variable which limits the width to a more comfortable size for reading.

<main>
  <p>This text will have an uncomfortably long line length.</p>
  <article>
    <p>This text will have a more suitable long line length.</p>
  </article>
</main>

It's possible to bypass this behaviour by wrapping the article tag inside a div tag:

<main>
  <div>
    <article>
      <p>This paragraph will take the full site width.</p>
    </article>
  </div>
</main>

This might be useful if you intend to split up the content inside the article using a multi-column layout. Note that the article should be the only tag inside the div tag or it will trigger another multi-column layout.
