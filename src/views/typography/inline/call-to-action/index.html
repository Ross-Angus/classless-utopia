Call to action

Call-to-action styles need to be flexible to allow them to be added almost anywhere in a document. Because of this requirement, the rel attribute was used to identify them.

In order to limit CSS resets, these styles have been scoped to children of the main element.

The rel attribute has specific semantic meaning which are occasionally updated. For example, the Internet Assigned Numbers Authority maintains a list of current valid strings. The list is long but only a few examples could be reused here.

Default state

Adding a rel attribute to an anchor tag will generate a default chevron to the right of the link text and invoke the button styling. However, it's recommended that the value of the rel attribute be set to related instead of being left empty. This will result in the styles below:
