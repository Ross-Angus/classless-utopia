Quotations

Inline quotations

Inline quotations can be marked up using the q tag: The q HTML element indicates that the enclosed text is a short inline quotation. Most modern browsers implement this by surrounding the text in quotation marks. This element is intended for short quotations that don't require paragraph breaks; for long quotations use the blockquote element.

Tooltips will display on q elements under the following conditions:

  • If the q tag has a cite attribute
    • ... and the user hovers their mouse over it
    • If the q tag has a tabindex attribute which allows it to fall into focus and the user puts the q tag into focus.

The tabindex attribute ...allows developers to make HTML elements focusable, allow or prevent them from being sequentially focusable (usually with the Tab key, hence the name) and determine their relative ordering for sequential focus navigation.

If a quotation does not have a cite attribute, no tooltip will display. If a quotation has an empty cite attribute, an empty tooltip will display
